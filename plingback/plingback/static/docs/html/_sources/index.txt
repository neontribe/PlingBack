Welcome to the PlingBack Documentation
======================================

Overview
--------

PlingBack provides different ways to solicit or give feedback about positive 
activities for young people which are collated and publicised through the 
Plings project at http://www.plings.net

This documentation covers the use and deployment of two example feedback widgets
and provides information which may be of interest to developers intending to 
submit data to the api or examine feedback which has been collected.
